Cache Strategy

Resource Cache Duration Reason
/robots.txt 86400s (24h) SEO crawlers cache, reduces bandwidth
/sitemap.xml 86400s (24h) Daily crawl index, safe to cache long-term
/rss 3600s (1h) Feed readers check hourly, balance freshness
/media/* 86400s (24h) Static assets, immutable filenames

Impact

  • ✅ SEO crawlers: cache robots/sitemap locally, fewer requests to origin
  • ✅ Bandwidth: 24h cache = ~95% reduction in crawler traffic
  • ✅ Browser caching: media now cached 24h (faster repeat visits)
  • ✅ CDN efficiency: Cloudflare can serve cached copy without origin hit
