Class Design

Cây kế thừa và quan hệ giữa các đối tượng. Hộp nét đứt = Abstract Base Class (ABC); mũi tên tam giác rỗng = kế thừa (is-a); ký hiệu ◆ = composition (has-a).

3. Service & Repository layer

ClassTrách nhiệm chính
AuthServiceĐăng nhập (verify PBKDF2, chống enumeration), phát hành token, audit cả thành công/thất bại
UserServiceCRUD user: business validation, check trùng, hash mật khẩu, UnitOfWork (user + audit nguyên tử)

ContentServiceTrang CMS theo slug (cache), sitemap entries, RSS items
SearchServiceTìm kiếm FTS5 (rank + prefix, fallback LIKE), chặn query < 2 ký tự
DashboardServiceGom metrics: đếm bảng, log theo level, health DB, cache stats, traffic
SystemServiceHồ sơ kết nối DB (mask DSN), tổng hợp BaseHealthChecker (server + database)
TrafficServicePage views + online: accumulator trong bộ nhớ (TrafficTracker), flush batch xuống DB, unique theo hash ẩn danh/ngày
SiteSettingsServiceKey-value cấu hình runtime (whitelist KNOWN_SETTINGS, cache 30s, maintenance mode)
ContactServiceForm liên hệ: honeypot drop thầm, throttle/IP, publish event contact.submitted
MediaServiceUpload: whitelist đuôi file, size cap, tên lưu trữ random hex (chặn traversal về cấu trúc)
BackupServiceVACUUM INTO snapshot + xoay vòng — job đêm và màn Backups dùng chung
RedirectServiceQuy tắc 301/302 cache 60s, resolve chỉ trên đường 404, auto-create khi đổi slug

6. Các class bổ sung (Batch 1–3)

Các sơ đồ SVG ở trên thể hiện bộ khung gốc; những class sau được bổ sung theo cùng đúng các pattern đó (mỗi dòng = subclass của base tương ứng + 1 dòng đăng ký ở bootstrap):

BaseClass bổ sung
AdminPageContentManagementPage (form editor) · ContactMessagesPage · MediaManagerPage · SessionManagerPage · BackupManagerPage · RedirectManagementPage · JobsMonitorPage · SettingsPage · AdminPasswordChangePage (Account Security + 2FA); UserManagementPage/MenuManagementPage có form editor + phân trang
PublicPageDynamicContentPage — route catch-all /{slug} (đăng ký cuối cùng) phục vụ content tạo từ CMS
AdminApiControllerContact · Media · Backup · Redirect · Settings (+ jobs/run và export CSV trong System/User/Log controllers)
BaseControllerMediaWebController (/media/{name}) · DynamicContentController
BaseSchedulerJobTrafficFlushJob · DatabaseBackupJob (ủy quyền BackupService)

UiComponentBarChartComponent (SVG server-side) · PaginationComponent
