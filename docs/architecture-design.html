Architecture Design

Kiến trúc phân tầng (Layered + Clean Architecture), một codebase triển khai linh hoạt theo module, dependency luôn hướng vào trong qua các Abstract Base Class.

1. Tổng quan stack

2. Sơ đồ kiến trúc phân tầng

Nguyên tắc cốt lõi: Domain không biết HTTP/SQL; Repository chỉ biết BaseDatabaseManager (không biết sqlite3); Service chỉ biết Repository + các ABC hạ tầng; Controller chỉ biết Service. Toàn bộ wiring cụ thể nằm duy nhất ở app/bootstrap.py (Composition Root).

HTTPS termination (profile proxy): container Caddy đứng trước toàn bộ hệ thống ở cổng 80/443 — tự xin và gia hạn SSL Let's Encrypt theo DOMAIN/ADMIN_DOMAIN, redirect HTTP→HTTPS, thêm HSTS. 6. Mô hình bảo mật

  • Mật khẩu: PBKDF2-HMAC-SHA256, salt ngẫu nhiên 16 byte, 310.000 vòng lặp, so sánh constant-time.
  • Token: HMAC-SHA256 ký + hết hạn + nhúng token_version — logout/đổi mật khẩu thu hồi NGAY mọi token cũ; Admin page dùng cookie HttpOnly + SameSite=Strict + Secure (production), API client dùng Authorization: Bearer.
  • Tài khoản seed bắt buộc đổi mật khẩu ở lần đăng nhập đầu (must_change_password) trước khi làm bất kỳ thao tác admin nào.
  • Content-Security-Policy với nonce sinh mới mỗi request — chỉ inline style/script do framework đóng dấu mới được chạy; frame-ancestors 'none'.
  • Login endpoint có rate-limit riêng (mặc định 5 lần / 5 phút / IP) chống brute-force.
  • RBAC: RoleGuard — mọi endpoint admin yêu cầu role admin; role đọc từ DB nên đổi quyền có hiệu lực tức thì.
  • SQL: 100% parameterized; cột ORDER BY qua whitelist từng repository (chống injection qua sort).
  • XSS: mọi giá trị động qua esc() / xml_escape — an toàn theo thiết kế.
  • Chống user-enumeration: sai user và sai mật khẩu trả cùng một thông báo.
  • Error sanitization: envelope lỗi chuẩn, không bao giờ lộ stack trace.
  • Rate limit API theo IP; security headers (X-Frame-Options DENY, nosniff, Referrer-Policy).
  • Trang admin noindex, nofollow; container chạy non-root user.
  • Tách module: server public không tồn tại route admin (404 tuyệt đối).
